### Crypti ###

Crypti is decentralized DApp platform.

Requires:

	* sqlite3 (version < 3.8.6)
	* nodejs
	* npm

## Installation

Clone repo and run: 

```sh
npm install --production
```

## Start

Go to crypti folder and run:

```sh
node app.js
```
